# AI-Creative-Toolkit
Model Organizer, Prompt Builder, and other tools all in one.

- MO tab is fully working.
- PB tab is working but needs some spacing adjustments.
The other tools are mostly placeholders for now. Some features work in their tabs but are not deemed ready for use.

- LoRA Info is my next goal. It will help save and organize info needed to use a LoRA.
- Prompt History will likely need a custom ComfyUI node to spit out a running log that this tab will read, allowing you to quickly save details that produce an image you like. Kind of like an advanced Prompt Builder.
- Workflows tab is intended to help find files you need to use a workflow obtained from someone else.
- Image Metadata kind of goes along with Prompt History, allowing you to read info saved in an image.
